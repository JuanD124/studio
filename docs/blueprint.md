# **App Name**: LavanderiaFacil

## Core Features:

- Price and Cost Input: Allow users to input the prices and costs for different types of garments or items to be washed.
- Item Storage and Retrieval: Implement a service where items can be stored and retrieved either by unique ID or customer name.
- Storage Duration Tracking: Track the storage duration for each item.
- Price and Item Modification: Enable modification of item prices, including adding new items to the price list.

## Style Guidelines:

- Primary color: Light sky blue (#87CEEB) to evoke cleanliness and freshness, relating to the cleaning of clothes. 
- Background color: Very light gray (#F0F0F0) for a neutral, clean backdrop.
- Accent color: Pale green (#98FF98) to signal order status changes
- Font pairing: 'Poppins' (sans-serif) for headlines and short descriptive text, and 'PT Sans' (sans-serif) for body text to balance modernity with readability. Note: currently only Google Fonts are supported.
- Use clean, simple icons for item categories (shirts, pants, etc.) and service types (wash, store).
- A clean, organized layout with clearly defined sections for pricing, storage management, and item details.
- Subtle transitions for actions such as adding an item or updating a price to give a fluid feel.
